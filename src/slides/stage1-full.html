Pipelines ğŸš€1

Two factions are fighting:

  • F# pipelines
  • Smart pipelines

Minimal proposal: sugar syntax

value |> f
// desugars to
f(value)

Desugaring also means that there's no need to bind object methods:

const count = 16
count |> 'na'.repeat
      |> (x => x + ' Batman!')

Wonderful with partial applications

player.score
  |> roundTo(?, 100)
  |> clamp(0, 1e6, ?)
